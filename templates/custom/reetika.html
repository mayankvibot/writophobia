Diabetic retinopathy grading system
(VIBOT Le Creusot FR, March 2022)
- The model aims to economize manual power and proposes a deep graph correlation network (DGCN) to develop automated diabetic retinopathy grading without any professional annotations.
- To evaluate the DGCN model, the model uses EyePACS-1 and Messidor-2 sets to perform grading results. It achieved an accuracy of 89.9% (91.8%), sensitivity of 88.2% (90.2%), and specificity of 91.3% (93.0%) on EyePACS-1 (Messidor-2) data set with a confidence index of 95% and commendable effectiveness on receiver operating characteristic (ROC) curve and t-SNE plots.
Vegetation segmentation in precision agriculture
(VIBOT Le Creusot FR, April 2022)
- Discriminating value crops from weeds is an important task in precision agriculture. This project explains the process of extracting features from multiple training images, fitting a classifier (e.g Random Forest or SVM), and segmenting images using a saved model.
- This project works on a feature based image segmentation using traditional machine learning ( multi-training images) where we get maximum accuracy of about 0.7 for the fifth Gabor feature.
Road plane extraction from stereo images
(VIBOT Le Creusot FR, April 2022)
- Road plane extraction can be helpful in detecting and avoiding objects for people having partial or no vision and also to perform object tracking.
- This was achieved by two methods, one by creating a ROI and performing Hough transform to detect the road correctly.
- The second method involves removing the common part from stereo images using Homography matrix where the internal parameters of the camnera are known.
